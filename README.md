## Installation

```bash
# Repository klonen
git clone https://github.com/DEIN_USERNAME/DEIN_REPO.git
cd DEIN_REPO

# Dependencies installieren
npm install

# Capacitor sync (Android-Dateien aktualisieren)
npx cap sync android
```
